MongdoDB Command 정리

Console상의 MongoDB에서 명령어를 알아보자

use [database]

  • DB 생성
  • 생성 후 생성된 DB를 사용하게 됨
  • 이미 존재하는 경우에 현존하는 데이터베이스를 사용
  • example

upbit라는 데이터베이스를 생성

db

  • 현재 사용중인 DB 확인
  • example

show dbs

  • 내가 만든 DB 리스트 확인
  • example
